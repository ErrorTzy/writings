型号：STC-89C52RC-40I-PDIP40

- 89：产品序列号
- C：工作电压
    + C：5.5-3.8V
    + LE：3.6-2.4V/3.4-2.4V
- 52：配置信息
    + 52：8kB ROM 512B RAM
    + 516：62kB ROM 1280B RAM
- 40：工作频率，40MHz
- I：工作温度范围
    + I：Industrial，-40-85
    + C：Commercial，0-70
- PDIP：一种封装类型
- 40：单片机管脚数

引脚有四组：P0，P1，P2，P3；每组有8根引脚

特殊引脚：

- VCC：电源正极
- GND：接地或电源负极
- RSI：复位
