 型号：STC-89C52RC-40I-PDIP40

- 89：产品序列号
- C：工作电压
    + C：5.5-3.8V
    + LE：3.6-2.4V/3.4-2.4V
- 52：配置信息
    + 52：8kB ROM 512B RAM
    + 516：62kB ROM 1280B RAM
- 40：工作频率，40MHz
- I：工作温度范围
    + I：Industrial，-40-85
    + C：Commercial，0-70
- PDIP：一种封装类型
- 40：单片机管脚数

引脚有四组：P0，P1，P2，P3；每组有8根引脚

特殊引脚：

- VCC：电源正极
- GND：接地或电源负极
- RSI：复位

### LED

- 正极引脚长，内电极小
- 负极引脚短，内电极长

![[Pasted image 20260529102338.png]]

- LED在导通时电阻极小，恒压电路中容易导致电流过大烧毁LED，因此需要串联限流电阻在导通时限制电流

![[Pasted image 20260529105840.png]]

LED由寄存器控制，0对应低电平，1对应高电平；如果引脚为低电平，则引脚起到负极作用，LED亮；如果引脚为高电平，则LED不亮

电阻标记法：

- 102：$10*10^{2}=1000\Omega$
- 473：$47*10^{3}=47000\Omega$

控制LED数码管时，由于一次只能控制一个（或多个相同LED），因此若要同时显示多个，那么就需要利用CPU时钟足够快的特性，快速依次显示不同LED

![[Pasted image 20260604143526.png]]

最简单的程序流程大约为：

- 选择LED1
- 设置P0显示x
- 选择LED2
- 设置P0显示y

但由于LED2被选择时，LED1依然具有高电平，因此LED1和2会同时显示x，导致错位。因此，需要加入关段和延迟的操作：

- 选择LED1
- 设置P0显示x
- sleep一段时间保持显示
- 设置P0不显示
- 选择LED2
- 设置P0显示y
- sleep一段时间保持显示
- 设置P0不显示
